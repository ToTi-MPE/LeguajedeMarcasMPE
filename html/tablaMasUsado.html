: Define el documento como un documento HTML. : Contiene metadatos sobre el documento, como el título y enlaces a hojas de estilo. : Establece el título del documento que aparece en la pestaña del navegador. <body>: Contiene el contenido principal del documento HTML. <h1>:a <h6>: Encabezados, donde `<h1>` es el más importante y `<h6>` el menos importante. <p>: Define un párrafo. <a>: Define un hipervínculo. <img>: Inserta una imagen. <ul> y <ol>: Listas no ordenadas y ordenadas, respectivamente. <li>: Define un elemento de lista. <div>: Define una división o sección en un documento. <span>: Define un contenedor en línea para contenido. <form>: Define un formulario para la entrada del usuario. <input>: Define un control de entrada dentro de un formulario. <button>: Define un botón clicable. <table>,<tr>,<td>: Utilizadas para crear tablas, filas y celdas, respectivamente.
