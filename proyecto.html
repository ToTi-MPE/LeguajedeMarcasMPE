Definición de Sistema Informático

Parte 1: Definición de Sistema Informático

Un sistema informático es un conjunto de componentes que interactúan entre sí para procesar, almacenar y transmitir información. Los componentes principales de un sistema informático son el hardware y el software, que trabajan de manera conjunta para ejecutar tareas y resolver problemas.

  • Hardware: Se refiere a los componentes físicos del sistema, como el procesador, la memoria, los dispositivos de almacenamiento, entre otros.
  • Software: Son los programas y sistemas operativos que controlan y permiten que el hardware ejecute las tareas necesarias.

Los sistemas informáticos están diseñados para ejecutar programas (software) que procesan datos y proporcionan resultados útiles para los usuarios.

Parte 1.1: Hardware

El hardware está compuesto por todos los dispositivos y componentes físicos que conforman un sistema informático. Estos dispositivos trabajan juntos para permitir la ejecución de programas y la interacción con el usuario.

1. Esquema de los Componentes de un Sistema Informático

Un sistema informático consta de los siguientes componentes esenciales:

  • CPU (Unidad Central de Procesamiento): Es el componente principal del sistema que se encarga de ejecutar las instrucciones de los programas y controlar el flujo de datos entre los diferentes componentes del sistema.
  • Memoria: Almacena los datos y las instrucciones que la CPU necesita para ejecutar las tareas.
  • Dispositivos de Entrada y Salida (E/S): Permiten la interacción entre el usuario y el sistema, tanto para recibir datos (entrada) como para enviarlos (salida).
  • Buses: Son los caminos a través de los cuales se transmiten los datos y las señales entre los componentes del sistema informático.

2. CPU (Unidad Central de Procesamiento)

La CPU es el "cerebro" del sistema informático, responsable de interpretar y ejecutar las instrucciones contenidas en los programas. Está encargada de las operaciones de cálculo y control necesarias para procesar la información.

Funciones de la CPU:

  • Ejecutar instrucciones: La CPU lee y ejecuta las instrucciones contenidas en los programas (como operaciones matemáticas, lógicas, etc.).
  • Controlar el flujo de información: La CPU coordina la comunicación entre otros componentes del sistema, como la memoria, las unidades de E/S y los dispositivos de almacenamiento.
  • Cálculos: Realiza cálculos aritméticos y lógicos necesarios para resolver los problemas planteados por los programas.

Partes de la CPU:

  1. ALU (Unidad Aritmético-Lógica): Se encarga de realizar las operaciones aritméticas (como sumas y multiplicaciones) y lógicas (como comparaciones de números).
  2. CU (Unidad de Control): La unidad de control supervisa la ejecución de instrucciones y coordina el flujo de información dentro de la CPU. Gestiona el ciclo de ejecución de las instrucciones.
  3. Registros: Son pequeñas áreas de almacenamiento de alta velocidad que se utilizan para almacenar temporalmente datos e instrucciones que están siendo procesados por la CPU.

3. Memoria

La memoria en un sistema informático es crucial porque almacena tanto los programas como los datos que son necesarios para el procesamiento. Sin memoria, la CPU no podría acceder a la información que necesita para ejecutar las instrucciones.

Funciones de la Memoria:

  • Almacenaje temporal: Guarda los datos que están siendo procesados de manera temporal.
  • Almacenaje permanente: Algunas memorias permiten almacenar información de manera permanente (no volátil), como el sistema operativo o archivos del usuario.
  • Acceso rápido: Las memorias cercanas a la CPU, como la caché y la RAM, permiten un acceso mucho más rápido que las de almacenamiento masivo, como los discos duros.

Jerarquía de la Memoria:

  1. Caché: Es una memoria extremadamente rápida que almacena datos e instrucciones que la CPU utiliza con frecuencia. Se encuentra dentro o cerca de la CPU. Existen diferentes niveles de caché (L1, L2, L3), con L1 siendo la más rápida pero de menor capacidad.
  2. RAM (Memoria de Acceso Aleatorio): Es la memoria principal del sistema y se utiliza para almacenar los programas y datos que están siendo ejecutados en un momento dado. Es volátil, lo que significa que pierde su contenido cuando se apaga el sistema.
  3. Almacenamiento Secundario: Incluye dispositivos como discos duros (HDD), unidades de estado sólido (SSD), y discos ópticos, que ofrecen una mayor capacidad de almacenamiento pero son más lentos en comparación con la memoria RAM o caché.

Ejemplos de Memoria:

  • Caché L1: Se encuentra directamente dentro de la CPU y es la más rápida, pero su capacidad es limitada.
  • RAM DDR4: Es un tipo de memoria dinámica utilizada en la mayoría de los ordenadores modernos, que ofrece una buena relación entre velocidad y capacidad.
  • SSD (Unidad de Estado Sólido): Almacena datos de manera permanente y es mucho más rápido que los discos duros tradicionales (HDD).

4. Unidades de Entrada y Salida

Las unidades de Entrada y Salida (E/S) son los dispositivos que permiten la interacción del usuario con el sistema informático. Las unidades de entrada permiten al usuario ingresar datos, mientras que las unidades de salida muestran los resultados o permiten al usuario recibir información.

Unidades de Entrada:

  • Teclado: Permite al usuario introducir texto y comandos.
  • Ratón: Dispositivo señalador que permite interactuar con el sistema gráfico.
  • Escáner: Convierte imágenes físicas en formato digital.
  • Micrófono: Captura sonido para ser procesado por el sistema.

Unidades de Salida:

  • Pantalla (Monitor): Muestra visualmente la información procesada por el sistema.
  • Impresora: Convierte la información digital en un formato físico.
  • Altavoces: Permiten emitir sonidos o música a partir de los datos procesados.

5. Buses

Un bus es un conjunto de cables y protocolos que permite la transmisión de datos entre diferentes componentes de un sistema informático. Los buses son cruciales para el funcionamiento de los sistemas, ya que sin ellos, la CPU no podría comunicarse con la memoria, las unidades de entrada/salida o el almacenamiento.

Tipos de Buses:

  1. Bus de Datos: Transporta los datos entre la CPU, la memoria y otros dispositivos periféricos. Este bus es bidireccional, lo que significa que los datos pueden viajar en ambas direcciones.
  2. Bus de Dirección: Transporta las direcciones de memoria a donde los datos deben ser leídos o escritos. Este bus es unidireccional, lo que significa que solo transmite datos de la CPU hacia los dispositivos de almacenamiento o memoria.
  3. Bus de Control: Transporta señales de control que gestionan el flujo de datos, como señales de lectura y escritura, o de inicio y parada de operaciones.

6. Esquema General de los Componentes y de Cómo se Conectan Entre Ellos

En un sistema informático, la comunicación entre los diferentes componentes se lleva a cabo mediante los buses. El CPU interactúa con la memoria para acceder a los datos y las instrucciones necesarias para ejecutar los programas. Además, las unidades de E/S permiten la interacción con el usuario, y los datos se transfieren entre todos estos componentes a través de los buses de datos, dirección y control.

Parte 1.2: Software

El software es el conjunto de programas, procedimientos y rutinas que dirigen y controlan el hardware del sistema informático para realizar tareas específicas. Existen diferentes tipos de software según su función y el nivel de interacción con el hardware.

1. Definición de Software y de Programa

Software: Se refiere a las aplicaciones, programas y sistemas operativos que son ejecutados por el hardware para realizar tareas específicas. El software se clasifica en dos grandes categorías:

  • Software de Sistema: Este software proporciona las herramientas necesarias para que el hardware y el software de aplicación trabajen juntos. Ejemplos incluyen el sistema operativo (como Windows, macOS, Linux), que gestiona los recursos del sistema y facilita la ejecución de programas.
  • Software de Aplicación: Son programas diseñados para realizar tareas específicas para el usuario. Ejemplos incluyen programas como Microsoft Word, Excel, navegadores web, y aplicaciones móviles.

Programa: Es un conjunto de instrucciones que le indican al sistema informático qué hacer. Un programa puede ser tan simple como una calculadora o tan complejo como un software de diseño gráfico o de gestión empresarial.

Clasificación de Software:

  1. Software de Sistema:
    • Sistema Operativo: Controla todos los aspectos del hardware y permite que otros programas funcionen. Ejemplos incluyen Windows, macOS y Linux.
    • Controladores de dispositivo: Son programas que permiten que el sistema operativo se comunique con el hardware específico, como impresoras, tarjetas gráficas y otros periféricos.
  2. Software de Aplicación:
    • Aplicaciones de Productividad: Ejemplos incluyen procesadores de texto como Microsoft Word y hojas de cálculo como Microsoft Excel.
    • Aplicaciones Gráficas y Multimedia: Como Adobe Photoshop para la edición de imágenes o VLC para reproducir medios.
    • Software de Navegación Web: Ejemplos incluyen navegadores como Google Chrome o Mozilla Firefox.

2. Definición de Algoritmo

Un algoritmo es un conjunto de pasos secuenciales bien definidos que resuelven un problema o realizan una tarea. Los algoritmos son fundamentales para cualquier programa de software, ya que describen cómo se deben procesar los datos para obtener un resultado.

Ejemplo de Algoritmo para Hacer un Sándwich de Mantequilla y Jamón Cocido Tostado:

  1. Tomar dos rebanadas de pan.
  2. Untar una capa uniforme de mantequilla sobre una de las rebanadas.
  3. Colocar una capa de jamón cocido sobre la mantequilla.
  4. Colocar la segunda rebanada de pan encima.
  5. Colocar el sándwich en una tostadora o plancha y tostar hasta que el pan esté dorado y crujiente.
  6. Sacar el sándwich de la tostadora o plancha.
  7. Cortar y servir.

Diagrama de Flujo (simplificado):

  • Inicio
  • Tomar pan
  • Untar mantequilla
  • Colocar jamón
  • Colocar segunda rebanada de pan
  • Tostar el sándwich
  • Servir
  • Fin

3. Definición de Sistemas de Numeración y Ejemplos Aplicados a la Informática

Los sistemas de numeración son conjuntos de símbolos y reglas para representar números. En informática, los sistemas más comunes son binario, octal y hexadecimal.

  • Binario (Base 2): Usa solo los dígitos 0 y 1. Es el sistema utilizado por los ordenadores internamente para realizar operaciones de procesamiento de información. Ejemplo: \( 1101_2 \) es igual a \( 13_{10} \) en decimal.
  • Octal (Base 8): Utiliza los dígitos 0-7 y es utilizado en algunos sistemas informáticos como una representación compacta del binario. Ejemplo: \( 15_8 \) es igual a \( 13_{10} \) en decimal.
  • Hexadecimal (Base 16): Utiliza los dígitos 0-9 y las letras A-F para representar los valores. Es muy usado en informática para representar números binarios de manera más compacta y legible. Ejemplo: \( A3_{16} \) es igual a \( 163_{10} \) en decimal.

4. Clasificación de Lenguajes de Programación

Los lenguajes de programación se dividen en distintas categorías según su cercanía al hardware o su nivel de abstracción:

  1. Lenguajes de Bajo Nivel:
    • Lenguaje Máquina: El lenguaje que entiende directamente el procesador, compuesto exclusivamente por secuencias de 0s y 1s.
    • Lenguaje Ensamblador: Un paso más cercano al lenguaje máquina pero usando mnemónicos para facilitar la programación humana. Cada instrucción ensambladora se corresponde con una operación en lenguaje máquina.
  2. Lenguajes de Alto Nivel:
    • Lenguajes Interpretados: El código se ejecuta línea por línea por un intérprete, sin necesidad de ser compilado previamente. Ejemplos: Python, JavaScript.
    • Lenguajes Compilados: El código fuente es traducido a código máquina por un compilador antes de ser ejecutado. Ejemplos: C, C++.
    • Lenguajes de Propósito General: Estos lenguajes pueden ser utilizados para una amplia variedad de aplicaciones. Ejemplos: Java, C#, Python.
    • Lenguajes de Programación Declarativa: Estos lenguajes expresan la lógica del cálculo sin describir el flujo de control. Ejemplos: SQL (para bases de datos) y Prolog.
